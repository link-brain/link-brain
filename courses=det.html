#HTML

أساسيات HTML لبناء هيكل صفحات الويب.

  • عناصر الهيكل (head, body, header, footer)
  • الروابط، الصور، القوائم والنماذج
  • العناصر الدلالية semantic elements
رؤية المصادر

#CSS

CSS لتصميم صفحات الويب وتخطيطها.

  • النماذج، الفليكس، جريد
  • الأنماط والألوان والخطوط
  • التجاوب media queries وresponsive design
رؤية المصادر

#JavaScript

لغة البرمجة لإضافة تفاعل للمواقع.

  • المتغيرات والدوال
  • DOM والتعامل مع الأحداث
  • البرمجة غير المتزامنة Promises وAsync/Await
رؤية المصادر

#Git

نظام التحكم في الإصدارات للتعاون وإدارة التغييرات.

  • الالتزامات Commits والفروع Branches
  • دمج التغييرات merge وpull requests
  • أدوات GitHub/GitLab للوصول والمراجعة
رؤية المصادر

#Frameworks

أطر العمل الشائعة لبناء واجهات متقدمة.

  • React: مكونات وحالة (state)
  • Vue: انسياب البيانات (reactivity)
  • Angular: بنية كاملة للتطبيقات
رؤية المصادر

#Build Tools

أدوات البناء وإدارة الحزم والتجميع.

  • npm وyarn لإدارة الحزم
  • Vite/Webpack لتجميع الموارد
  • أتمتة المهام والأداء
رؤية المصادر

#Design

مبادئ التصميم وتجربة المستخدم.

  • تصميم واجهة المستخدم UI
  • نظام الألوان والطباعة
  • استخدام مكتبات مثل Tailwind أو Bootstrap
رؤية المصادر

#Deployment

نشر التطبيقات على الإنترنت ومفاهيم CI/CD.

  • Vercel وNetlify للنشر السريع
  • مبادئ CI/CD
  • إعدادات الدومين والشهادات SSL
رؤية المصادر

#Performance

تحسين سرعة الموقع وتقليل زمن التحميل.

  • تحسين الصور وتقليل الطلبات
  • التخزين المؤقت Caching وLazy Loading
  • تحسين CSS وJS وتقليل الحجم
رؤية المصادر

#Accessibility

جعل المواقع سهلة الاستخدام للجميع ووفق معايير WCAG.

  • السمات ARIA
  • تباين الألوان والقراءة
  • التنقل عبر لوحة المفاتيح

#Testing

طرق اختبار الأكواد وضمان الجودة.

  • اختبار الوحدات Unit Tests
  • اختبار التكامل Integration Tests
  • أدوات مثل Jest وCypress
